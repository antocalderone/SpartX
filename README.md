## Tecnologie e Librerie Utilizzate

Questo progetto è costruito utilizzando le seguenti tecnologie e librerie open-source:

*   **Flutter**: Il framework UI di Google per la creazione di applicazioni native per mobile, web e desktop da un'unica base di codice.
*   **Dart**: Il linguaggio di programmazione utilizzato da Flutter.

### Librerie Open-Source

*   [**cupertino_icons**](https://pub.dev/packages/cupertino_icons): Fornisce le icone in stile iOS.
*   [**file_picker**](https://pub.dev/packages/file_picker): Consente di selezionare file dal dispositivo.
*   [**flutter_pdfview**](https://pub.dev/packages/flutter_pdfview): Un visualizzatore di PDF per Flutter.
*   [**path_provider**](https://pub.dev/packages/path_provider): Per trovare percorsi di uso comune nel filesystem.
*   [**uuid**](https://pub.dev/packages/uuid): Per generare UUID (Universally Unique Identifiers).
